# 奇门遁甲排盘工具

一个基于网页的奇门遁甲排盘工具，支持真太阳时计算、自动定位等功能。

## 功能特点

- 📅 **干支历法计算**：自动计算年、月、日、时四柱
- 🌞 **真太阳时支持**：根据地理位置计算真太阳时
- 📍 **自动定位**：支持获取当前位置或手动选择城市
- 🎯 **奇门排盘**：
  - 自动计算节气、三元、局数
  - 支持阴阳遁和九局选择
  - 显示值符、值使信息
- 🗂️ **综合盘展示**：
  - 九星（天盘）
  - 八门
  - 八神
  - 地盘
  - 暗干
  - 空亡、马星标记
- 📖 **详细过程**：可展开查看排盘过程的详细信息

## 在线使用

访问：https://leighann.github.io/qimen/

## 本地使用

直接在浏览器中打开 `index.html` 文件即可使用。

## 技术栈

- 纯前端实现，无需后端服务
- [lunar-javascript](https://github.com/6tail/lunar-javascript) - 农历计算库
- [SunCalc](https://github.com/mourner/suncalc) - 太阳位置计算库
